# Monitoring and Observability

## Metrics Collection

The system exposes Prometheus-compatible metrics for monitoring.

### Key Metrics

- Flow processing rate (flows/sec)
- ML inference latency
- LLM API response times
- Alert generation rate

## Grafana Dashboards

Pre-built dashboards for system monitoring and threat visualization.

## Health Checks

Each agent exposes health check endpoints for Kubernetes liveness and readiness probes.
